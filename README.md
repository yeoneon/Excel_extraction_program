# Python Windows App Project

This is a modern Windows application built with Python and **CustomTkinter**.

## Features
- **Modern UI**: Uses CustomTkinter for a Windows 11 look and feel.
- **Dark/Light Mode**: User-selectable appearance modes.
- **Fast Development**: Minimal boilerplate and easy-to-use widgets.

## Prerequisites
- Python 3.10 or higher.

## Setup Instructions
1. **Create and Activate Virtual Environment**:
   ```powershell
   python -m venv .venv
   .venv\Scripts\activate
   ```
2. **Install Dependencies**:
   ```powershell
   pip install customtkinter
   ```
3. **Run the Application**:
   ```powershell
   python main.py
   ```

## Project Structure
- `main.py`: Entry point and UI logic.
- `.venv/`: Python virtual environment (ignored by git).
